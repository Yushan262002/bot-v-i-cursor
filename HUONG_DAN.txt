========================================
HUONG DAN SU DUNG BOT - 3 BUOC DON GIAN
========================================

BUOC 1: CAI DEPENDENCIES
-------------------------
Mo PowerShell trong thu muc bot, chay:

    pip install flask ccxt python-dotenv


BUOC 2: TAO FILE .ENV
----------------------
1. Copy file .env.example thanh .env:

    copy .env.example .env

2. Mo file .env, sua credentials:

    OKX_API_KEY=pk-xxxxx-xxxxx        <- API key that
    OKX_API_SECRET=sk-xxxxx-xxxxx     <- Secret that
    OKX_API_PASSWORD=your_password    <- Password that

3. Luu lai (Ctrl+S)


BUOC 3: CHAY BOT
-----------------
    python bot.py

LOI THUONG GAP
========================================

Loi: "ModuleNotFoundError: No module named 'flask'"
Fix: pip install flask ccxt python-dotenv

Loi: "KeyError: 'OKX_API_KEY'"
Fix: Kiem tra file .env ton tai va co du credentials

Loi: "ccxt.AuthenticationError"
Fix: API credentials sai, kiem tra lai .env

Loi: "Port 5000 already in use"
Fix: Doi port trong bot.py dong cuoi: app.run(port=5001)
